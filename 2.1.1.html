2 Перспективные форматы

2.1.1 Новые технологии

VVC включает множество инновационных технологий кодирования:

  1. Квадратичная древовидная структура с тройным разбиением: более гибкое разделение блоков
  2. Множественные типы трансформации: DCT-II, DST-VII, DCT-VIII для различных типов контента
  3. Улучшенное внутрикадровое предсказание: 67 угловых режимов и новые неугловые режимы
  4. Аффинная компенсация движения: учет вращения, масштабирования и сдвига объектов
  5. Bilateral matching: двустороннее сопоставление для улучшения предсказания
  6. Адаптивное петлевое фильтрование: ALF для минимизации артефактов сжатия

VVC революционизирует традиционный подход к видеокодированию через внедрение адаптивных алгоритмов, способных анализировать характеристики контента в реальном времени. Система множественных типов трансформации позволяет автоматически выбирать наиболее подходящий математический аппарат для каждого блока изображения, что особенно эффективно для сложных текстур и переходных областей.

Аффинная компенсация движения представляет качественный скачок в обработке динамических сцен, позволяя точно отслеживать вращение, масштабирование и деформацию объектов. Это критично для современного контента с активной камерной работой, спортивных трансляций и игрового видео.
