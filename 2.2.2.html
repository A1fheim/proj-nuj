2 Перспективные форматы

2.2.2 Базовый и основной профили

EVC предлагает гибкий подход к лицензированию:

Baseline Profile

  1. Использует только технологии, доступные в H.264
  2. Отсутствие новых патентных ограничений
  3. Эффективность сравнима с H.264 High Profile
  4. Подходит для бюджетных проектов

Main Profile

  1. Включает новые технологии для повышения эффективности
  2. Умеренные патентные обязательства
  3. До 25% лучше H.264 по эффективности
  4. Конкурентоспособен с H.265 в базовых сценариях

Двухуровневая архитектура EVC отражает реалии современного видеорынка, где сосуществуют массовые приложения с жесткими ограничениями по стоимости и премиум-сегмент с требованиями к максимальной эффективности. Baseline Profile обеспечивает безопасный переход для разработчиков, опасающихся патентных рисков, в то время как Main Profile предлагает современные возможности с контролируемыми обязательствами.

Возможность динамического переключения между профилями в рамках одного потока открывает новые сценарии использования, где критичные сегменты видео могут кодироваться в Main Profile для максимального качества, а остальной контент - в Baseline для минимизации рисков.
