1 Основные современные форматы

1.2.1 Архитектура кодека

HEVC использует более гибкую структуру блоков по сравнению с H.264:

  1. Coding Tree Units (CTU): размером до 64x64 пикселей
  2. Квадратичная древовидная структура: рекурсивное разделение блоков
  3. Prediction Units (PU): для предсказания движения и внутрикадрового предсказания
  4. Transform Units (TU): для применения дискретного косинусного преобразования

Революционным нововведением HEVC стала возможность адаптивного выбора размера кодируемых единиц от 4x4 до 64x64 пикселей в зависимости от сложности изображения. Это позволяет использовать большие блоки для однородных областей (небо, стены) и мелкие блоки для детализированных участков (текстуры, лица).

Иерархическая структура разделения блоков в H.265 обеспечивает оптимальный баланс между точностью представления и эффективностью сжатия, автоматически подстраиваясь под локальные характеристики изображения без необходимости предварительного анализа контента.
